# Pixel Brawler

A 2D browser fighting game where you battle waves of armed mobs (sword, scythe, and gun enemies) and duel a named rival boss every 5th wave — built with Street Fighter/Tekken-style combos, jump-attack special moves, and finisher animations.

No installs needed: open `index.html` in any browser, or play it live via GitHub Pages.

## About

You play an unarmed pixel-art fighter surviving an endless gauntlet of enemies. Chain light and heavy attacks into stackable combos, launch enemies with a Rising Uppercut, dash in with an Aerial Rush, and unleash a meter-charged special or ultimate finisher. Every kill can end in a slow-motion finisher animation. Health bars: yours on the left, the enemy's on the right.

**Controls:** `A`/`D` move · `W` jump · `S` crouch/block · `L` light attack · `K` heavy attack · `J` special/ultimate · double-tap `A`/`D` to dash · jump + `K`/`L` for aerial specials.

Built entirely in vanilla HTML5 Canvas/JavaScript — no game engine, no external art assets, no audio.
